INTRODUCTION
------------

The Username Policy module allows site administrators to configure policies for
username, so that every user will have an altered username generated by module.

 * For a full description of the module visit:
   https://www.drupal.org/project/username_policy

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/username_policy


REQUIREMENTS
------------

This module is not dependent on any other modules.


INSTALLATION
------------

 * Install the module as you would normally install a contributed
   Drupal module. Visit https://www.drupal.org/node/1897420 for further
   information.


CONFIGURATION
-------------

 *  Set permissions at
      http://yoursite.com/admin/people/permissions#module-username_policy
 * The administrative user interface can be found at
      http://yoursite.com/admin/config/people/username_policy
 * Visit http://yoursite.com/admin/help/username_policy for help.
